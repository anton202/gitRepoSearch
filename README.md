# GitRepoSearch

# How to run the project

1. clone the repo.
2. in the command line navigate to the cloned folder.
3. cd to client.
4. cd to gitRepoSearch.
5. run npm i.
6. run ng build.
7. navigate back to the cloned folder.
8. cd to server.
9. run npm i.
10. run node server.js.
11. open browser at localhost:8080.
